## Project Structure

The project is structured into two main directories:

- `backend`: Contains the Spring Boot backend files.
- `frontend`: Houses the React frontend files.

## Setup Instructions

### Backend (Spring Boot)

1. Navigate to the `backend` directory.
2. Open the project in your preferred Java IDE (e.g., STS, IntelliJ IDEA).
3. Build and run the Spring Boot application.
4. The backend will run on `http://localhost:8003`.

### Frontend (React)

1. Navigate to the `frontend` directory.
2. Install dependencies using npm or yarn: `npm install` or `yarn install`.
3. Start the React development server: `npm start` or `yarn start`.
4. The frontend will be accessible on `http://localhost:3000`.
